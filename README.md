# yeahtheb
Up to your imagination to figure out what these scripts do :)

## Dependencies:
	- argv
	- BeautiulSoup4
	- requests
	- wget
	- yaml

## Commands (Schedulers):
	- a = add a url
 	- s = save list into a cache file
 	- l = load cache
 	- lp = load cache with current list kept in RAM.
 	- d = run through list and download
